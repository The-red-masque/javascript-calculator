# JavaScript Calculator
A calculator using JavaScript, html, and css.
  - $\exists !$ numpad button set
  - $\exists !$ operator button set
  - $\exists !$ clear history button
 - $\forall$ \(Key of numpad\) 
    - there is a button with a string of the number.
    - the button has class: number-button

 - $\forall$ \(Key of operators\) 
    - there is a button with a string of the operator.
    - the button has class: operator-button
  - $input \iff clickOnButton \lor keyPress$.
